FAQ

자주 묻는 질문

전하영님은 어떤 엔지니어인가요?+
교육 현장에서 문제를 정의하고 시스템으로 해결해 온 경험을 바탕으로, 데이터 분석과 RAG·LLM 에이전트로 실제 동작하는 제품을 만드는 AI/ML 엔지니어입니다.
어떤 직무를 희망하나요?+
AI/ML 엔지니어, 데이터 엔지니어, 데이터 분석가, 프롬프트 엔지니어 직무로 구직 중입니다.
주요 기술 스택은 무엇인가요?+
Python·FastAPI·LangChain/LangGraph·RAG·PyTorch·Pandas·SQL·Docker·React/TypeScript를 사용합니다.
강점은 무엇인가요?+
데이터 이면의 근본 원인을 끝까지 추적하는 구조적 분석력과, 문제 정의부터 배포까지 전체 파이프라인을 연결하는 실행력입니다.
포트폴리오/코드는 어디서 보나요?+
GitHub(github.com/vosnuev)에서 전체 저장소를, 위 프로젝트 카드에서 라이브 데모를 확인하실 수 있습니다.
